# MBS — 6 Separate User Flows (Concept Note v2 + Fresher/Intern)

Six distinct learners from Class 5 to first job, each with its **own question bank, own behaviour-first flow, adaptive paths, and embedded validity controls**. Drawn from `MBS_v3_3_FullQuestionBank.xlsx` (1016 items, incl. communication/negotiation/conflict/teamwork/decision/EQ/entrepreneurship tests); game modules referenced by code.

## User 1 — Early Learner  ·  Class 5
*Surface intrinsic traits via play — curiosity, social style, thinking preference, early motivation. No career prediction.*

- **Duration:** 30 min  ·  **Target items:** 30  ·  **Delivery:** Fully gamified narrative; ZERO Likert. Picture forced-choice, revealed-preference exploration, gentle economic & risk games.

**Flow:**
- **0 Story warm-up** — Avatar & guide (-, n=0)
- **1 Curiosity** — Explore vs follow-the-path (creativity, n=4); What would you do next? (big5, n=2)
- **2 Thinking style** — Picture puzzles (apt_abstract, n=4); Sorting game (apt_spatial, n=2)
- **3 Social style** — Play alone or with friends (disc, n=4); Sticker-sharing game (motivation, n=2)
- **4 Sensory & attention** — Go/No-Go tap (game:M11 Go/No-Go (inhibition), n=3); Steady Trace (game:M22 Steady Trace, n=3)
- **5 Risk & motivation** — Balloon (gentle) (-, n=2); What makes you happy? (motivation, n=4)

**Adaptive paths:**
- *pacing:* Auto-pace by engagement speed; narrate all items.
- *difficulty:* Easy-only; 2 errors -> easier item.
- *no_high_stakes:* No scores shown to child; persona output only.
- *wellness:* Only severe signals -> counsellor (never the child).

**Bias / validity controls:**
- *telemetry:* Per-item response latency, long-string & answer-change logging on every block.
- *attention_checks:* Embedded attention-check + infrequency items; careless responders flagged.
- *validity_band:* Each report carries a confidence band (high / interpret-with-caution).
- *scoring:* Forced-choice scored with Thurstonian-IRT; behavioural/game scores weighted above self-report; methods fused by confidence, not naive mean.
- *note:* No self-report at all; everything behavioural or forced-choice.

**Report:** Learner persona (avatar + metaphor), strength bars, home+school tips, alert zones, NEP portfolio box.

## User 2 — Middle Learner  ·  Class 8
*Academic behaviour, emerging aptitude, motivation, real learning style, focus/fatigue, decision style.*

- **Duration:** 60 min  ·  **Target items:** 50  ·  **Delivery:** Behaviour-first: games + forced-choice + SJTs; minimal Likert; AI-scored short answers.

**Flow:**
- **1 Emerging aptitude** — Verbal reasoning (apt_verbal, n=6); Logic & patterns (apt_abstract, n=6); Visual / spatial (apt_spatial, n=4)
- **2 Achievement motive** — Ring-toss / dart distance (motivation, n=6)
- **3 Real learning style** — Format Lab (learn 4 ways, quiz each) (learning, n=6)
- **4 Focus vs fatigue** — Sustained attention (game:M10 Split Track (divided attention), n=3); Speed tap (processing_speed, n=2)
- **5 Study discipline** — Study-habit scenarios (softskills, n=5)
- **6 Behaviour (DISC)** — This-or-that pairs (disc, n=8)
- **7 Decision style** — Two Doors (clue cards + timer) (decision, n=4)

**Adaptive paths:**
- *difficulty:* <50% first 2 -> easier; >=80% first 3 -> stretch.
- *time_extension:* Slow-but-accurate get more time.
- *shortening:* High confidence -> shorten section.
- *ai_scored:* Short answers scored by Claude Haiku.

**Bias / validity controls:**
- *telemetry:* Per-item response latency, long-string & answer-change logging on every block.
- *attention_checks:* Embedded attention-check + infrequency items; careless responders flagged.
- *validity_band:* Each report carries a confidence band (high / interpret-with-caution).
- *scoring:* Forced-choice scored with Thurstonian-IRT; behavioural/game scores weighted above self-report; methods fused by confidence, not naive mean.

**Report:** Emerging-aptitude radar, learning-format fit, achievement-motive profile, attention/fatigue map, stream-fit predictors, parent summary.

## User 3 — Stream Selector  ·  Classes 9-10
*First high-stakes tool -> ranked STREAM recommendation (PCM/PCB/Commerce/Humanities/Hybrid). Faking pressure is real, so tilt behavioural.*

- **Duration:** 120 min  ·  **Target items:** 85  ·  **Delivery:** Adaptive aptitude games + culture-fair items + forced-choice personality + interest sims + validity scale.

**Flow:**
- **S1 Aptitude (adaptive, culture-fair)** — Verbal reasoning (apt_verbal, n=12); Numerical reasoning (apt_numerical, n=12); Spatial / logical (apt_spatial, n=8); Mechanical / technical (apt_mechanical, n=8); Abstract (Raven-style) (apt_abstract, n=6)
- **S2 Interests & style (behaviour-first)** — Career Theme-Park (ride dwell-time) (riasec, n=12); This-or-that style pairs (disc, n=12); People/Data/Ideas/Things sort (motivation, n=6)
- **S3 Validity & stream-fit** — Subject micro-sims (softskills, n=4); Validity scale (SD + infrequency) (survey, n=3)

**Adaptive paths:**
- *difficulty:* Per-module easy/standard/stretch.
- *shortening:* Cut section 20-30% on high confidence.
- *stream_routing:* Aptitude+interest+motivation triangulated -> rank streams.
- *mismatch_alert:* Flag interest-aptitude conflict.
- *stereotype_guard:* No demographic prompts before aptitude.

**Bias / validity controls:**
- *telemetry:* Per-item response latency, long-string & answer-change logging on every block.
- *attention_checks:* Embedded attention-check + infrequency items; careless responders flagged.
- *validity_band:* Each report carries a confidence band (high / interpret-with-caution).
- *scoring:* Forced-choice scored with Thurstonian-IRT; behavioural/game scores weighted above self-report; methods fused by confidence, not naive mean.
- *social_desirability:* SD scale; if high, lean on game scores & show caution band.

**Report:** Ranked streams, strength-subject map, compatibility chart, mismatch alert, parent guide.

## User 4 — Career Explorer  ·  Classes 11-12
*Full-spectrum map to 20 career domains (4 groups) + fit & next steps. Now incl. communication, EQ, decision, teamwork/conflict, values/meaning.*

- **Duration:** 180 min  ·  **Target items:** 130  ·  **Delivery:** Behaviour-first: forced-choice personality, interest sims, domain simulations, economic games, AI-scored reflection. New v3.3 tests delivered as SJTs/sims.

**Flow:**
- **0 Intake** — Context & consent (survey, n=6)
- **1 Personality (behaviour-first)** — Big Five via forced-choice (big5, n=20); Conscientiousness in-tray / tidy-desk (game:M4 Environment Simulator, n=0); DISC pairs (disc, n=10)
- **2 Interests** — Career Theme-Park (riasec, n=12); People/Data/Idea/Thing sort (motivation, n=6)
- **3 Aptitude / multiple intelligence** — Verbal & critical (apt_verbal, n=12); Numerical & analytical (apt_numerical, n=10); Spatial & visual (apt_spatial, n=8); Abstract (Raven) (apt_abstract, n=10)
- **4 Work-readiness tests (NEW v3.3)** — Communication SJT (comm, n=6); EQ read-the-room (eq, n=6); Decision style — Two Doors (decision, n=6); Team & conflict micro-sims (teamwork, n=4); Conflict-handling SJT (conflict, n=4); Values & meaning (philosophy, n=4)
- **5 Career-domain fit (ADAPTIVE)** — Day-as-a-___ (top 5-6 domains) (strategy_career, n=6)
- **6 Wellness & reflection** — Wellness screener (wellness, n=6); Failure-reflection & aspiration (survey, n=4)
- **S3 Games (router)** — Cognitive/creative games (game:M5 Cognitive Lab, n=0); ICT games (>=2 T) (game:T1 Logic Flow, n=0); Knowledge games (>=3 K) (game:K3 Ecosystem Architect, n=0)

**Adaptive paths:**
- *session3_routing:* Router fires modules by RIASEC top-3 + ability (min 2 T + 3 K + matched M).
- *domain_firing:* Only top-cluster domain sims shown.
- *difficulty:* Per-module easy/standard/stretch.
- *wellness_flag:* PHQ/GAD/SI + AI distress -> counsellor + iCall.
- *ai_scoring:* Open items scored by Claude.

**Bias / validity controls:**
- *telemetry:* Per-item response latency, long-string & answer-change logging on every block.
- *attention_checks:* Embedded attention-check + infrequency items; careless responders flagged.
- *validity_band:* Each report carries a confidence band (high / interpret-with-caution).
- *scoring:* Forced-choice scored with Thurstonian-IRT; behavioural/game scores weighted above self-report; methods fused by confidence, not naive mean.
- *overclaiming:* Over-claiming module in knowledge section catches exaggeration.
- *behaviour_first:* Personality/interests/work-readiness delivered as forced-choice/SJT/sim, not Likert.

**Report:** Top-5 domains + fit, full 20-domain matrix, communication/EQ/decision/teamwork scorecard, 'why this fits', next steps, self-awareness quotient.

## User 5 — Undergraduate / Career-Readiness  ·  Undergraduate
*Domain fit PLUS employability: communication, negotiation, conflict, teamwork, decision, EQ, learning agility, entrepreneurship, self-concept, work orientation; job vs founder path.*

- **Duration:** 190 min  ·  **Target items:** 150  ·  **Delivery:** Simulation-heavy: in-tray, negotiation & team sims, economic games, Trade-off Island, internship evidence; minimal Likert. All new v3.3 tests as sims/SJTs.

**Flow:**
- **0 Intake & evidence** — Context & consent (survey, n=6); Internship/project upload (-, n=2)
- **1 Personality** — Big Five forced-choice (big5, n=18); DISC pairs (disc, n=10)
- **2 Interests & work orientation** — Career Theme-Park (riasec, n=12); Work Styles Arena (workstyles Work Styles Arena, n=10); Trade-off Island (values) (workvalues, n=8); Environment Simulator (context) (workcontext Environment Simulator, n=6)
- **3 Work-readiness sims (NEW v3.3)** — In-tray / in-basket (game:M4 Environment Simulator, n=0); Communication & tone (comm, n=8); Negotiation sim (negotiation, n=8); Conflict resolution SJT (conflict, n=8); Team-chat collaboration sim (teamwork, n=8); Decision under ambiguity (decision, n=8); EQ read-the-room (eq, n=8)
- **4 Employability & skills** — Tech skills + Expert Challenge (game:M7 Expert Challenge, n=6); Soft-skill SJTs (softskills, n=8); Learning agility (Format Lab) (learning_agility, n=8); Career adaptability (4Cs) (strategy_career, n=8)
- **5 Domain, aspiration & self-concept** — Day-as-a-___ (top domains incl. founder) (strategy_career, n=6); Entrepreneurial orientation (entrepreneur, n=8); Self-concept clarity (intrapersonal, n=6); Aspiration / influences / challenge (survey, n=4)
- **6 Integrity & risk** — Honest Dice (-, n=0); Balloon Vault (-, n=0)
- **7 Wellness** — Wellness screener (wellness, n=6)
- **S3 Games (router)** — ICT games (role-relevant) (game:T4 Spreadsheet Sprint, n=0); Knowledge games (domain) (game:K2 Market Mover, n=0)

**Adaptive paths:**
- *experience_routing:* Internship AI tags bias domain firing & feedback.
- *path_branch:* Forks into JOB-readiness or ENTREPRENEURSHIP path by behaviour+orientation.
- *shortening:* Aptitude shortened where school data imported.
- *session3_routing:* Router weighted to work readiness.
- *wellness_flag:* Distress -> counsellor + iCall.
- *ai_scoring:* Internship, aspiration, reflection, sims scored by Claude.

**Bias / validity controls:**
- *telemetry:* Per-item response latency, long-string & answer-change logging on every block.
- *attention_checks:* Embedded attention-check + infrequency items; careless responders flagged.
- *validity_band:* Each report carries a confidence band (high / interpret-with-caution).
- *scoring:* Forced-choice scored with Thurstonian-IRT; behavioural/game scores weighted above self-report; methods fused by confidence, not naive mean.
- *behaviour_first:* Most fakeable population -> simulate the most; self-report only for wellness.
- *evidence_weighting:* Internship upload + sims weighted above any remaining self-report.

**Report:** Top domains + role fit, employability scorecard (comm/negotiation/conflict/teamwork/decision/EQ/learning-agility/integrity), adaptability index, job vs entrepreneurship path, next-step plan, wellness note.

## User 6 — Fresher / 1st-Year / Intern  ·  Entry-level workforce / interns
*On-the-job readiness for a first role: how they actually behave at work — communication, collaboration, conflict, negotiation, decision-making under ambiguity, EQ, learning agility, integrity, stress, ownership; plus role fit & 30-60-90 plan.*

- **Duration:** 150 min  ·  **Target items:** 130  ·  **Delivery:** Simulation-FIRST. A day-in-the-job runs through in-tray, team chat, a tough conversation, a decision under ambiguity, an ethics moment. Self-report only for wellness.

**Flow:**
- **0 Intake & evidence** — Role/context & consent (survey, n=4); Resume / internship upload (-, n=2)
- **1 The workday simulation (core)** — In-tray: triage your day-1 inbox (game:M4 Environment Simulator, n=0); Day-as-a-new-joiner branching (strategy_career, n=6)
- **2 Communication & EQ** — Write/choose the right message (tone) (comm, n=10); Read-the-room (manager/peer cues) (eq, n=8)
- **3 Collaboration, conflict & negotiation** — Team-chat collaboration sim (teamwork, n=8); Tough conversation / conflict (conflict, n=8); Negotiate scope/deadline with NPC (negotiation, n=8)
- **4 Decision-making & problem-solving** — Decision under ambiguity (Two Doors) (decision, n=8); Spot the Break (problem sensitivity) (game:M20 Spot the Break, n=0); Crisis Commander (prioritise under pressure) (game:K9 Crisis Commander, n=0)
- **5 Ownership, learning agility & integrity** — Learning agility (Format Lab + adapt) (learning_agility, n=8); Self-management / ownership SJT (softskills, n=8); Honest Dice (integrity) (-, n=0); Entrepreneurial / intrapreneur orientation (entrepreneur, n=6)
- **6 Work orientation & fit** — Work Styles Arena (workstyles Work Styles Arena, n=8); Trade-off Island (work values) (workvalues, n=6); Work context preferences (workcontext, n=6)
- **7 Confirmatory aptitude (short)** — Numerical/analytical (apt_numerical, n=6); Verbal/critical (apt_verbal, n=6); Abstract (apt_abstract, n=6)
- **8 Wellness** — Wellness screener (wellness, n=6)
- **S3 Games (router)** — ICT games (role tools, e.g. Spreadsheet/Phishing) (game:T4 Spreadsheet Sprint, n=0); Domain knowledge games (game:K2 Market Mover, n=0)

**Adaptive paths:**
- *role_routing:* Resume/role tags + interests fire the relevant domain sims, ICT (T) and knowledge (K) games.
- *path_branch:* Outputs an employee-readiness path AND an intrapreneur/entrepreneur signal.
- *difficulty:* Per-module easy/standard/stretch; aptitude shortened if UG data imported.
- *wellness_flag:* Distress -> counsellor + iCall.
- *ai_scoring:* Inbox, messages, negotiation, reflection scored by Claude.

**Bias / validity controls:**
- *telemetry:* Per-item response latency, long-string & answer-change logging on every block.
- *attention_checks:* Embedded attention-check + infrequency items; careless responders flagged.
- *validity_band:* Each report carries a confidence band (high / interpret-with-caution).
- *scoring:* Forced-choice scored with Thurstonian-IRT; behavioural/game scores weighted above self-report; methods fused by confidence, not naive mean.
- *behaviour_first:* ~85% behavioural/sim/SJT; self-report only for wellness.
- *evidence_weighting:* Sim performance + internship evidence dominate the composite.

**Report:** Work-readiness scorecard (communication, collaboration, conflict, negotiation, decision-making, EQ, learning agility, integrity, stress, ownership), role/domain fit, strengths & dev areas, a 30-60-90 day plan, and a job-vs-entrepreneurship signal.
